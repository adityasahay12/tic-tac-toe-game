# 🎮 Tic-Tac-Toe Game

## 🚀 Overview
This is a simple and interactive Tic-Tac-Toe game built using JavaScript. The project focuses on implementing core game logic and providing a clean and responsive user interface.

## 🔥 Features
- Two-player game (Player X and Player O)
- Win detection logic
- Draw detection
- Reset game functionality
- Responsive UI design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 How to Play
1. Open the game in browser
2. Player X starts first
3. Players take turns marking cells
4. First to align 3 marks wins
